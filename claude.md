# 프로젝트 개요
이 프로젝트는 학생들이 직관적으로 이해하고 플레이할 수 있는 웹 기반 교육용 게임입니다.

# 기술 스택
- 프론트엔드: Vanilla JavaScript, HTML5, CSS3
- 데이터베이스/서버: Firebase (설정 시 databaseURL 등 누락 주의)
- 배포: GitHub Pages

# 클로드 코드(AI)를 위한 필수 지침
1. **코드 스타일:** 초보자도 이해하기 쉽도록 변수명은 직관적으로 짓고, 주요 게임 로직에는 반드시 상세한 한글 주석을 달아주세요.
2. **안전한 수정:** 기존 코드를 수정하기 전에 반드시 현재 디렉토리의 구조와 대상 파일을 먼저 읽고(read) 분석하세요.
3. **UI/UX 기준:** 화면은 큼직하고 시원하게 구성하며, 사용자 상호작용(클릭, 정답/오답 등)에 즉각적인 피드백(애니메이션 등)을 제공하세요. 
4. **문제 해결:** 터미널 에러가 발생하면 로그를 스스로 확인하고, 원인을 분석한 뒤 수정 방안을 제안하세요.
